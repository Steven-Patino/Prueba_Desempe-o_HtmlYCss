#Prueba_Desempe-o_HtmlYCss

**how to clone a repository**

-Open Terminal.
-Change the current working directory to the location where you want the cloned directory.
-Type git clone https://github.com/Steven-Patino/Prueba_Desempe-o_HtmlYCss.git
-Press Enter to create your local clone.

**how to view the project**

-Open the cloned repository in Visual Studio Code and download the Live Server extension.
-In the PDD_M2.html file, right-click and look for "open with live server".
